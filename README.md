# sprichwort2console

Get funny sayings from [http://sprichwortgenerator.de/](http://sprichwortgenerator.de/) into your command line.

## How to use

Clone this repo with
```sh
git clone git@github.com:timlapluie/sprichwort2console.git
```
Step into the directory
```sh
cd sprichwort2console
```
and execute
```
./sprichwort.rb
```
